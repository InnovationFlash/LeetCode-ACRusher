## 简单题，经典栈思路；

# 题目概述：
直接栈思路，很典型，判断弹栈出栈即可；

# 具体代码：
```C++
class Solution {
public:
    bool isValid(string s) {
        stack<char>st;
        for (int i = 0; i < s.size(); i++) {
            if (st.empty() || s[i] == '(' || s[i] == '{' || s[i] == '[') {
                st.push(s[i]);
            }
            else if (!st.empty() && (s[i] == ')' && st.top() == '(' || s[i] == ']' && st.top() == '[' || s[i] == '}' && st.top() == '{')) {
                st.pop();
            }
            else {
                return false;
            }
        }
        if (!st.empty())
            return false;
        return true;
    }
};
```
